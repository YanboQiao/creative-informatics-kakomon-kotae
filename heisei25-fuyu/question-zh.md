# 2013年冬季入学考试

### 东京大学信息理工学系研究科 创造信息学专攻

## 编程

令小写字母 a 到 z 为布尔变量。运算符 `&` 表示逻辑与，`+` 表示逻辑或，且 `&` 的优先级高于 `+`。给定一个由这些运算符和变量构成的布尔公式，若对每个变量的一组赋值使公式求值为真，则该赋值被称为该公式的一个「解」。布尔公式以单行文本表示，文本中不包含空格或制表符，仅由变量名和运算符构成。

例如，当给定的公式为：

```
b&a+b&c+a&b&c
```

则赋值 `a=true, b=true, c=false` 是一个解。

(1) 编写程序读取布尔公式，并以 `+` 为分隔符拆分公式。例如输入上面的公式时，程序需输出：

```
b&a
b&c
a&b&c
```

(2) 编写程序读取布尔公式并求其所有解。若存在解则输出全部解；若不存在解则输出 “none”。

(3) 在 (2) 的程序基础上扩展以支持否定运算符 `!`。该运算符为前缀一元运算，且具有最高优先级。例如程序必须能够接受以下公式：

```
!a&b&!c+a&!d
```

(4) 在 (3) 的程序基础上继续扩展，使其能够接受包含括号 `(` 与 `)` 的公式。

(5) 编写程序读取符合 (4) 规格的布尔公式，并输出与其等价的析取范式（DNF）。DNF 是由变量（每个变量仅出现一次）、`&`、`!` 组成的子句，通过析取连接而成。以下公式是一个 DNF：

```
a&b&c+a&b&!c+!a&b&c
```

(6) 编写程序读取符合 (4) 规格的布尔公式，并输出与其等价的合取范式（CNF）。CNF 是由变量（每个变量仅出现一次）、`+`、`!` 组成的子句，通过合取连接而成。以下公式是一个 CNF：

```
(!a+b+!c)&(!a+b+c)&(a+!b+c)&(a+b+!c)&(a+b+c)
```
