**ceil = 向上取整**
**floor = 向下取整**


# math.ceil(x)

**返回大于或等于 x 的最小整数。**
也就是 **往上凑** 到整数。

例子：

| 表达式        | 结果 | 说明           |
| ---------- | -- | ------------ |
| ceil(2.1)  | 3  | 往上凑          |
| ceil(2.0)  | 2  | 已经是整数，不变     |
| ceil(-1.2) | -1 | -1 比 -1.2 更大 |

---

# **2. math.floor(x)**

**返回小于或等于 x 的最大整数。**
也就是 **往下凑** 到整数。

例子：

| 表达式         | 结果 | 说明           |
| ----------- | -- | ------------ |
| floor(2.9)  | 2  | 往下凑          |
| floor(2.0)  | 2  | 已经是整数，不变     |
| floor(-1.2) | -2 | -2 比 -1.2 更小 |
