这些都是 **Python 字符串（`str`）的布尔判断方法**，返回值全是 `True` 或 `False`。下面用 **中文+简洁例子** 全部讲解。

---

# 1. `str.isalnum()`

判断字符串是否只包含 **字母或数字**。

```python
"a123".isalnum()     # True
"abc!".isalnum()     # False
"".isalnum()         # False 空字符串永远 False
```

---

# 2. `str.isalpha()`

判断是否只包含 **字母**（不能包含数字）。

```python
"abc".isalpha()      # True
"abc1".isalpha()     # False
```

---

# 3. `str.isascii()`

判断是否所有字符都是 ASCII（0–127），即英文字符、数字、基本符号。

```python
"abc123".isascii()     # True
"你好".isascii()        # False
```

---

# 4. `str.isdecimal()`

判断是否只包含 **十进制数字**（严格：0–9）。

```python
"123".isdecimal()     # True
"１２３".isdecimal()   # False（全角数字也算 False）
```

---

# 5. `str.isdigit()`

判断是否只包含数字字符，范围比 `isdecimal()` 更宽（包含全角数字等）。

```python
"123".isdigit()       # True
"１２３".isdigit()     # True
"Ⅲ".isdigit()         # True
```

---

# 6. `str.isidentifier()`

判断字符串能否做 Python 变量名。

```python
"abc".isidentifier()      # True
"_x123".isidentifier()    # True
"1abc".isidentifier()     # False
"hello-world".isidentifier() # False
```

---

# 7. `str.islower()`

判断字符串是否全部是 **小写字母**（必须包含字母）。

```python
"abc".islower()       # True
"abc123".islower()    # True
"Abc".islower()       # False
```

---

# 8. `str.isnumeric()`

判断是否是数字字符（范围最广），比 `isdigit()` 更宽，包括罗马数字等各种数字符号。

```python
"123".isnumeric()       # True
"１２３".isnumeric()     # True
"Ⅳ".isnumeric()         # True
```

---

# 9. `str.isprintable()`

判断是否全部可打印（不能含控制字符）。

```python
"hello!".isprintable()     # True
"\n".isprintable()         # False
```

---

# 10. `str.isspace()`

判断是否都是空白字符（空格、`\n`、`\t` 等）。

```python
"   ".isspace()        # True
"\t\n".isspace()       # True
"a ".isspace()         # False
```

---

# 11. `str.istitle()`

判断是否为“标题格式”：每个单词首字母大写，其它小写。

```python
"Hello World".istitle()     # True
"Hello world".istitle()     # False
```

---

# 12. `str.isupper()`

判断是否全部为大写字母（可包含数字）。

```python
"ABC".isupper()       # True
"ABC123".isupper()    # True
"AbC".isupper()       # False
```

---

# 总览总结表

| 方法               | 说明            |
| ---------------- | ------------- |
| `isalnum()`      | 全字母或数字        |
| `isalpha()`      | 全字母           |
| `isascii()`      | ASCII 范围内     |
| `isdecimal()`    | 严格十进制数字       |
| `isdigit()`      | 数字字符（更宽）      |
| `isnumeric()`    | 数字字符（最宽）      |
| `isidentifier()` | 合法 Python 变量名 |
| `islower()`      | 全小写           |
| `isupper()`      | 全大写           |
| `isspace()`      | 空白字符          |
| `isprintable()`  | 可打印字符         |
| `istitle()`      | 标题格式（每词首字母大写） |

全部解释结束。
